# Panduan Kontribusi (Contributing Guide)

Terima kasih telah tertarik untuk berkontribusi pada **PSSM**! Kami sangat menghargai bantuan Anda untuk membuat pendidikan digital di Indonesia menjadi lebih baik.

## Bagaimana Anda Bisa Membantu?
1.  **Melaporkan Bug:** Gunakan GitHub Issues untuk melaporkan bug yang Anda temukan. Sertakan langkah-langkah untuk mereproduksi bug tersebut.
2.  **Saran Fitur:** Kami selalu terbuka untuk ide-ide baru. Silakan ajukan melalui GitHub Issues.
3.  **Pull Requests:** 
    - Lakukan fork pada repository ini.
    - Buat branch baru untuk fitur atau perbaikan bug Anda (`git checkout -b feature/nama-fitur`).
    - Pastikan kode Anda mengikuti standar PSR-12 (untuk PHP).
    - Lakukan commit dengan pesan yang jelas.
    - Ajukan Pull Request ke branch `main`.

## Standar Pengembangan
- Gunakan **Laravel Best Practices**.
- Pastikan setiap fitur baru memiliki validasi yang kuat.
- Jika mengubah UI, pastikan tetap responsif dan mengikuti gaya **Shadcn UI**.
