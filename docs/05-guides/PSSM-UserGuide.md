# PSSM - User Guide
## Panduan Pengguna Platform Smart School Management v1.0

**Untuk:** Guru, Siswa, Ketua Kelas, Admin Sekolah  
**Bahasa:** Indonesia  
**URL:** https://pssm.school

---

## 1. Memulai PSSM

### 1.1 Login

1. Buka **https://pssm.school** di browser (Chrome/Firefox/Safari)
2. Masukkan **Email** dan **Password** yang diberikan admin
3. Klik **"Masuk"**
4. Jika pertama kali login, Anda akan diminta **verifikasi email** — cek inbox Anda

> ⚠️ Password minimal 12 karakter, harus mengandung huruf besar, kecil, angka, dan simbol.

### 1.2 Lupa Password

1. Klik **"Lupa Password?"** di halaman login
2. Masukkan email Anda
3. Cek inbox email → klik link reset password
4. Buat password baru
5. Login dengan password baru

### 1.3 Logout

Klik **nama Anda** di pojok kanan atas → **"Keluar"**

> ⚠️ Sesi otomatis berakhir setelah 2 jam tidak aktif.

## 2. Panduan untuk GURU

### 2.1 Dashboard Guru

Setelah login, Anda akan melihat:
- **Kelas Anda** — Daftar kelas yang Anda ampu
- **Tugas Pending Review** — Berapa tugas yang belum dinilai
- **Kuis Aktif** — Kuis yang sedang berlangsung

### 2.2 Membuat Tugas Baru

1. Klik **"+ Buat Tugas"** di dashboard
2. Isi formulir:
   - **Judul** — Nama tugas (wajib)
   - **Deskripsi** — Instruksi untuk siswa
   - **Kelas** — Pilih kelas tujuan
   - **Mata Pelajaran** — Pilih mapel
   - **Deadline** — Tanggal dan jam terakhir pengumpulan
   - **Nilai Maksimum** — Default: 100
   - **Lampiran** — Upload file PDF/Word/gambar (maks 10MB)
3. Klik **"Publish"** untuk langsung kirim ke siswa
4. Siswa akan mendapat **notifikasi email** otomatis

### 2.3 Menilai Tugas Siswa

1. Buka tugas yang ingin dinilai
2. Klik tab **"Submissions"**
3. Klik nama siswa untuk melihat jawaban mereka
4. Download file yang dikumpulkan (jika ada)
5. Isi **Nilai** (0 sampai nilai maksimum)
6. Isi **Feedback** (opsional — saran perbaikan)
7. Klik **"Simpan Nilai"**
8. Siswa akan mendapat notifikasi bahwa tugasnya sudah dinilai

> 💡 **Tips:** Gunakan fitur **"AI Feedback"** untuk essay — AI akan menganalisis struktur, grammar, dan memberikan saran otomatis.

### 2.4 Membuat Kuis Manual

1. Klik **"+ Buat Kuis"**
2. Isi info kuis:
   - **Judul**, **Kelas**, **Mata Pelajaran**
   - **Durasi** — Berapa menit siswa mengerjakan
   - **Waktu Mulai & Berakhir** — Kapan kuis bisa diakses
3. Tambah soal satu per satu:
   - **Pilihan Ganda** — 4 opsi (A-D), pilih jawaban benar
   - **Essay** — Siswa menulis jawaban bebas
   - **True/False** — Benar atau Salah
4. Klik **"Publish"**

### 2.5 Membuat Kuis dengan AI 🤖

1. Klik **"+ Buat Kuis"** → pilih tab **"Generate dengan AI"**
2. Masukkan:
   - **Topik** — contoh: "Fotosintesis pada Tumbuhan"
   - **Jumlah Soal** — 5 sampai 20
   - **Tingkat Kesulitan** — Mudah, Sedang, Sulit
3. Klik **"Generate"** — tunggu 10-15 detik
4. **Review** semua soal yang dihasilkan AI
5. Edit jika perlu → klik **"Publish"**

> ⚠️ **Selalu review** soal AI sebelum dipublish — AI bisa menghasilkan soal yang kurang tepat.

### 2.6 Melihat Analytics

1. Klik **"Analytics"** di sidebar
2. Pilih kelas dan rentang waktu
3. Lihat:
   - **Rata-rata nilai** per tugas
   - **Siswa dengan nilai tertinggi/terendah**
   - **Submission rate** — berapa % siswa yang mengumpulkan tugas
   - **Topik tersulit** — tugas/kuis dengan rata-rata terendah

### 2.7 Export Nilai ke Excel

1. Buka **Analytics** → pilih kelas
2. Klik **"Export"** → pilih format **Excel (.xlsx)**
3. File akan otomatis terdownload
4. Buka file di Microsoft Excel atau Google Sheets

### 2.8 Rekap Absensi

1. Klik **"Absensi"** di sidebar
2. Pilih **kelas**, **mata pelajaran**, dan **tanggal**
3. Centang status setiap siswa: **Hadir / Izin / Sakit / Alpha**
4. Klik **"Simpan"**

## 3. Panduan untuk SISWA

### 3.1 Dashboard Siswa

Setelah login, Anda akan melihat:
- **Tugas Deadline Terdekat** — Daftar tugas yang harus dikumpulkan
- **Nilai Rata-rata** — Rata-rata nilai Anda
- **Notifikasi** — Tugas baru, nilai keluar, pengumuman

### 3.2 Mengerjakan Tugas

1. Klik **"Lihat Semua Tugas"** atau klik tugas dari dashboard
2. Baca instruksi dari guru
3. Download lampiran dari guru (jika ada)
4. Kerjakan tugas, lalu:
   - **Upload file** — PDF, Word, atau gambar (maks 10MB)
   - **Tulis jawaban** — Untuk essay, langsung ketik di kolom yang tersedia
5. Klik **"Submit Tugas"**

> ⚠️ Anda bisa **re-submit** selama tugas belum dinilai dan belum melewati deadline.  
> ⚠️ Submit setelah deadline akan ditandai **"Terlambat"**.

### 3.3 Mengerjakan Kuis

1. Buka kuis yang tersedia dari dashboard
2. Klik **"Mulai Kuis"** — timer akan berjalan
3. Jawab semua soal
4. Klik **"Submit"** sebelum timer habis
5. Untuk pilihan ganda — nilai keluar **langsung**
6. Untuk essay — guru akan menilai nanti

> ⚠️ Jika timer habis, jawaban Anda akan otomatis di-submit.  
> ⚠️ Anda hanya bisa mengerjakan kuis sesuai jumlah percobaan yang diberikan guru.

### 3.4 Melihat Nilai

1. Klik **"Nilai"** di sidebar
2. Lihat nilai per mata pelajaran
3. Klik tugas/kuis tertentu untuk melihat **feedback** dari guru
4. Lihat grafik perkembangan nilai Anda dari waktu ke waktu

### 3.5 Notifikasi

Anda akan mendapat notifikasi via:
- **Di website** — bell icon di pojok kanan atas
- **Email** — untuk tugas baru dan nilai keluar
- **WhatsApp** (jika diaktifkan) — reminder deadline H-1

## 4. Panduan untuk KETUA KELAS

Ketua kelas memiliki semua fitur siswa, **ditambah:**

### 4.1 Input Absensi

1. Klik **"Absensi"** di sidebar
2. Pilih mata pelajaran dan tanggal
3. Centang kehadiran teman-teman sekelas
4. Klik **"Simpan"**

### 4.2 Broadcast Pengumuman

1. Klik **"Pengumuman"** → **"+ Buat Pengumuman"**
2. Isi judul dan konten pengumuman
3. Pilih prioritas: Normal, Penting, Urgent
4. Klik **"Kirim"** — semua teman sekelas akan mendapat notifikasi

## 5. Panduan untuk ADMIN SEKOLAH (Super Admin)

### 5.1 Dashboard Admin

- **Total Siswa, Guru, Kelas** — Statistik keseluruhan sekolah
- **Grafik Kehadiran** — 30 hari terakhir
- **Submission Rate** — Berapa % tugas dikumpulkan
- **Top 5 Siswa Terbaik** — Berdasarkan nilai rata-rata

### 5.2 Manajemen User

1. Klik **"Users"** di sidebar
2. Untuk **menambah user baru**:
   - Klik **"+ Tambah User"**
   - Isi nama, email, role (Guru/Siswa/Ketua Kelas)
   - User akan mendapat email undangan untuk set password
3. Untuk **import massal siswa**:
   - Klik **"Import CSV"**
   - Upload file CSV dengan format:
   ```
   nama,email,nis,kelas
   Budi Santoso,budi@school.id,12345,11 IPA 1
   Dewi Lestari,dewi@school.id,12346,11 IPA 1
   ```
4. Untuk **menonaktifkan user**: klik user → **"Nonaktifkan"**

### 5.3 Manajemen Kelas

1. Klik **"Kelas"** di sidebar
2. **Buat kelas baru**: nama, tingkat (10/11/12), tahun ajaran, wali kelas
3. **Assign siswa**: pilih dari daftar atau import CSV
4. **Assign mapel & guru**: tentukan siapa mengajar apa di kelas mana

### 5.4 Export Data

1. Klik **"Reports"** di sidebar
2. Pilih jenis report:
   - **Rekap Nilai** seluruh kelas → Excel
   - **Raport Siswa** → PDF
   - **Rekap Absensi** → Excel/CSV
3. Pilih filter (kelas, semester, rentang waktu)
4. Klik **"Export"**

### 5.5 Activity Log (Audit Trail)

1. Klik **"Logs"** di sidebar
2. Lihat semua aktivitas: siapa, kapan, melakukan apa
3. Filter berdasarkan user, tipe aktivitas, atau tanggal
4. Berguna untuk investigasi masalah atau memonitor penggunaan

## 6. FAQ (Pertanyaan Umum)

### Umum

**Q: Browser apa yang didukung?**  
A: Chrome, Firefox, Safari, Edge versi terbaru. Tidak support IE.

**Q: Bisa diakses dari HP?**  
A: Ya, website responsive — bisa diakses dari HP, tablet, atau laptop.

**Q: Data saya aman?**  
A: Ya. Kami mengenkripsi data sensitif, menggunakan HTTPS, dan melakukan backup harian.

### Guru

**Q: Berapa lama AI generate kuis?**  
A: Sekitar 10-15 detik untuk 10 soal.

**Q: Bisa edit kuis setelah dipublish?**  
A: Bisa, selama belum ada siswa yang mengerjakan.

**Q: Format file apa yang diterima untuk lampiran tugas?**  
A: PDF, Word (.doc/.docx), dan gambar (JPG, PNG). Maksimum 10MB.

### Siswa

**Q: Bisa submit ulang tugas yang sudah dikumpulkan?**  
A: Bisa, selama belum dinilai guru dan belum lewat deadline.

**Q: Timer kuis habis tapi belum selesai?**  
A: Jawaban yang sudah diisi akan otomatis ter-submit.

**Q: Kenapa tidak bisa mengerjakan kuis?**  
A: Kemungkinan: kuis belum dibuka, sudah lewat waktu, atau sudah mencapai batas percobaan.

## 7. Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Tidak bisa login | Cek email/password. Gunakan "Lupa Password" jika perlu. |
| Halaman loading lama | Refresh browser (Ctrl+F5). Cek koneksi internet. |
| File gagal upload | Pastikan file < 10MB dan format PDF/Word/JPG/PNG. |
| Nilai tidak muncul | Guru mungkin belum menilai. Cek kembali nanti. |
| Notifikasi tidak masuk | Cek folder spam di email. Pastikan email benar di profil. |
| Sesi tiba-tiba logout | Sesi expire setelah 2 jam inaktif. Login kembali. |
